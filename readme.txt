Simple Tiny Slider

== Description ==

This plugin is a great sliding JavaScript slideshow can easily be customized to integrate with 
any website using CSS. You can add content to it, not images, and it degrades gracefully without
 JavaScript support. The script supports automatic rotation with the option to auto-resume, 
 an asset class on a navigation list, if applicable, and a tilt direction (vertical or horizontal).


This plugin implements the slideshow from JavaScript is Mechael Leibeger.
